Release VB-2019-04

Submitted by rwieck on Tue, 2019-04-09 11:15
We are pleased to announce the April 2019 release of VectorBase (VB-2019-04):
  • New species Leptotrombidium deliense - host vector for scrub typhus. Full details this mite genome are available in the publication "Genomes of trombidid mites reveal novel predicted allergens and laterally transferred genes associated with secondary metabolism." Dong et al, Gigasciene 2018.
  • In this release there were updates to the gene sets for 4 species based on community annotation ( Aedes aegypti, An. arabiensis, An. farauti, and An. gambiae. ).
  • New summary tracks have been added in Apollo as a concise compilation of all splice site predictions from our aligned RNA-Seq data. There are two main types of tracks: one "known" (contain splice junctions in accord with the gene model) and several "unknown" (those come with various coverage threshold). Each species also has a merged bigwig file for the coverage.
  • The variation mart for SNP and indel data will be retired from this release as originally announced in the October 2018 release notes. The variation BioMart has been replaced with the variation search facility which is described in this tutorial.
  • There is a new view in the PopBio map that visualizes the blood meal hosts of vectors.
  • Lots of new Population Biology data, including mosquito surveillance data from Iowa going back to the 1970s
  • The Genotype Explorer application will be retired in the next (June 2019) VectorBase release.
  • New tutorials includes these six topics: Tour, Aedes & Culex, Genome Browser (includes 6 videos/demos), Anopheles, Query & Browse (Search, BioMart, BLAST, Clustal, Hmmer -includes 6 videos/demos) and Biomphalaria.
