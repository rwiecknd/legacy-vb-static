Release VB-2014-12

Submitted by lawson on Tue, 2014-12-16 09:44

We are pleased to announce the December release of VectorBase (VB-2014-12):

  • The expression database has been "spring cleaned" to improve consistency of p-values. It also now has ontology-enhanced search.
  • Population Biology - several projects have been updated to improve data consistency and/or search.
  • Gene sets have been updated this release for: Rhodnius prolixus, Glossina austeni, Glossina brevipalpis, Glossina fuscipes and Glossina pallidipes
  • NewRNAseq sets have been added for: Lutzomyia longipalpis (x8), Glossina austeni (x2), Musca domestica (x5) and Biomphalaria glabrata (x8)
  • 146 Anopheles gambiae RNA genes that lacked external references were associated with Rfam and/or mirBase identifiers. Statistics on the coverage of pairwise alignments are now available.
  • The Blastx genomic similarity search against the combined VectorBase proteome that was initially created for use in Web Apollo is now also displayed in VectorBase. The data can be viewed as a track in the genome browser via the "protein alignments" section of the display configuration.

Genomes

VectorBase is committed to a new release every two months with all data freely available for public use based on NIH/NIAID policy. A list of these changes and the state of current versions on this date (e.g., current gene sets) can be found on the Releases section of VectorBase.

Expression Data

Some older experiments within VectorBase's expression resource had issues with technical replicates (such as dye swaps) being treated the same as biological replicates. This was artificially improving p-values for those experiments. From this release onwards, only biological replicates are counted as replicates. Now if it says n=3 in the gene expression report pages, this always means that three biological replicates were analysed.

Expression data search has been improved through the use of ontology terms, particularly via the Advanced Search form. Now you can search using broad concepts (such as "alimentary") in the high (or low) conditions box, and get results items annotated with more fine-grained terms (such as "gastric caeca").

Population Biology/Insecticide Resistance

Various improvements have been made at the database level to several PopBio projects. For instance, the publications linked to Malaria Atlas Project samples are now searchable by their authors and titles (previously it was only possible to search by PubMed IDs). Also, the entire PopBio database has been updated to use the correct species names following the split of Anopheles gambiae s.s. into Anopheles coluzzii and Anopheles gambiae s.s..
