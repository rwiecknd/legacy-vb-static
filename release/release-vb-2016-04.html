Release VB-2016-04

Submitted by creinkin on Tue, 2016-04-12 08:56

We are pleased to announce the April 2016 release of VectorBase (VB-2016-04):

  • New organism Cimex lectularius (Harlan strain)
  • New Aedes aegypti gene expression profiling data for three flavivirus infection time courses (Dengue, West Nile and Yellow Fever viruses)
  • New mass spectrometry data for Anopheles gambiae and Ixodes scapularis
  • Updated InterPro domain annotation of all VectorBase genomes
  • InterPro domain annotation of all transcriptome assemblies
  • New date search in the PopBio map
  • A single nucleotide polymorphism (SNP) calling workflow added to Galaxy
  • New password expiry requirements
  • We have restructured and greatly improved the "About" page. It is design to give current and new users a quick guide of what can be done with the site, including some sample queries with analyses, and much more
  • The "Search" tutorial has been updated, the material includes practice exercises with answer key, and new video with its slides
  • We now have a YouTube Channel were we will add video tutorials. This time we invite you to watch "Tour", which is a (6 min) overview of the site navigation, data types, tools and resources

New password expiry requirements

VectorBase users who have an account will be required to update the password every six months. Thirty (30) days before that an email will be sent to account holders to create a new password, the email will have the necessary links and instructions. You do not need to update your Web Apollo password, this is a separate account.

In this release we have added the Harlan strain of Cimex lectularius (the bed bug). Protein features for all species have been updated, using InterPro version 55.0.

Expression data

In this release we have added a new experiment profiling gene expression in response to several different flavivirus infections.

Population Biology/Insecticide Resistance

Four insecticide resistance papers have been added:

  • Fonseca-Gonzlez, I., Quiones, ML., Lenhart, A., Brogdon WG (2011) Insecticide resistance status of Aedes aegypti (L.) from Colombia
  • Hammad, M., Ahmed, N., Khan, IA., Shah, B. Kan, A., Ali, M., Rasheed, MT., Junaid, K., Adnan, M., Zaki, AB., Ahmed, S (2015) Studies on the efficacy of selected insecticides against Anopheles mosquitoes of village Goth Bhoorji (Sindh) Pakistan
  • Ocampo, CB., Salazar-Terreros, MJ., Mina, NJ., McAllister, J., Brogdon W (2012) Insecticide resistance status of Aedes aegypti in 10 localities in Colombia
  • Sahu, SS., Gunasekaran, K., Vijayakumar, T., Jambulingam P (2015) Triple insecticide resistance in Anopheles culicifacies: a practical impediment for malaria control in Odisha State, India

These are all available in the PopBio map which now has newly added date and normalized insecticide resistance value search functionality.
