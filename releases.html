Releases

VectorBase is committed to a release every two months with all data freely available for public use based on NIH/NIAID policy. A list of these changes and the state of current versions are found here.

Title/Description Release date
Release VB-2019-08

PopBio new data and map interface improvements, new sandfly and mosquito variation datasets, Ensembl support of all 43 VectorBase genomes, updated splice junction files in Apollo, improved assemblies for 21 Anopheles available from downloads, updates to three FAQs and the creation of a new one related to CSIRO

 August 28, 2019
Release VB-2019-06

three new assemblies (A. coluzzii, A. funestus, I. scapularis), L. deliense RNA gene predictions, gene metadata submissions for A. aegypti and A. gambiae, projection of variant calls in A. funestus, new RNAseq tracks for 25 species, PopBio fixes and improvements, three 3 Apollo videos and a new version of the workshop handout with 'Critical Thinking' sections

 June 25, 2019
Release VB-2019-04

new mite genome, updates to 4 gene sets, Apollo updates, variation query migrating from BioMart to Search, Genotype explorer to be retired, new PopBio view and data, six new tutorials

 April 24, 2019
Release VB-2019-02

gene set updates, RNAseq depth statistics now available, Apollo to remain open but discontinuing patch builds, variation support reminders, differential expression updates, new PopBio data & map bug fixes and improvements, Apollo update, 2019 outreach calendar

 February 19, 2019
Title/Description Release date
Release VB-2018-12

support for the Aedes aegypti L3 genome assembly will be discontinued, new PopBio features and new type of datasets, outreach calendar for 2019 open, Apollo to temporary stop receiving annotations, new tutorials and documentation, help desk closed for the holidays.

 December 12, 2018
Release VB-2018-10

gene set updates for 24 species, new variation data for A. gambiae, new differential expression data for A. aegypti (infected with dengue and Zika), BioMart variation to be retired, new PopBio data and map interface improved, new PopBio and tour tutorials, upcoming outreach events include webinars and workshops

 October 25, 2018
Release VB-2018-08

new PopBio pathogen view, new PopBio abundance data, new page metrics, PhyloCFS track retired from genome browser, Search for variation data has been added, new variation tutorial, new outreach events

August 28, 2018
Release VB-2018-06

gene set and assembly updates, Aedes variation data, updated code for Apollo in VB code site, retirement of support for AaegL3 assembly, BLAST/Apollo integration, new gene knowledge cards, PopBio map new features.

June 28, 2018
Release VB-2018-04

new PopBio, variation and expression data; improved PopBio features, new whole genome alignments, new tutorials and videos.

April 24, 2018
Release VB-2018-02

Updates to 11 gene sets, new variation data from Ag1000 AR3, updated whole genome alignments, updated variation effect predictions, expression microarray probes updated, Apollo for the new A. aegypti genome, many new features for Advanced Search and PopBio, new tutorials.

 February 21, 2018
Title/Description Release date
Release VB-2017-12

New A. aegypti and A. atroparvus assemblies, new A. gambiae variation data, new STRUCTURE export option, new Find a Data Display resource in the Genome Browser, new expression and PopBio data sets.

 December 11, 2017
Release VB-2017-10

New PopBio abundance view and share link; gene sets updates due to manual annotations for 30 species; new variation, RNAseq, insecticide resistance and genotype data; BLAST improvements, and 13 tutorial updates.

October 25, 2017
Release VB-2017-08

A species gene set update, 2 large popbio datasets updates, 3 differential expression experiments, 5 insecticide resistance studies, new tools features and capabilities, updated Submission page.

 August 29, 2017
Release VB-2017-06

4 differential expression experiments, 10 new PopBio studies, updates to protein data (alignments and domains), 12 species geneset updates, public REST API, a new tool and improvements to current tools.

 June 27, 2017
Release VB-2017-04

This release has updated genesets for 12 species, 4 gene expression datasets, 3 popbio, and 11 insecticide resistance publications.

 April 25, 2017
Release VB-2017-02

This release has updated RNA gene predictions for all species, 4 gene expression datasets and 5 insecticide resistance publications.

 February 21, 2017
